♙ Tosho Library ♙

Durant cette année 2024/2025 notre professeur de dev Mme Tonon, nous a lancé sur un projet dit bibliothèque.
Comme son nom l'indique, c'est une bibliothèque où nous devions référencer 10 oeuvres de notre choix.
Maintenant, rentrons dans le coeur du sujet, le code !
Nous devions commencer par coder en HTML/CSS, pour avoir un bon aperçu de ce que nous pouvions faire. Quelques mois après, nous avions pour consigne de tout basculer en PHP qui est un langage de programmation côté serveur, pour commencer à se pencher vers la gestion d'une base de donnée.
Toute cette partie, on appelle ça du "no code" par ce qu'on ne code pas comme son nom l'indique. Pour se faire, nous avons utilisé phpMyAdmin, c'est un outil logiciel libre qui permet de gérer facilement les bases de données MySQL via une interface web.
